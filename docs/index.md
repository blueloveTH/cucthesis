## 使用说明

本模板有两种使用方式，本地编译和Overleaf。

#### Overleaf

Overleaf是一个网页版的LaTeX集成编辑环境，我们已经将模板副本上传至Overleaf，你只需注册一个账号，就能立即开始写作。

#### 本地编译

本地编译需要配置环境，LaTeX和C一样，是一门语言。因此它需要**编译器**和**编辑器**两个组件。

+   编译器取决于你的操作系统，Windows上可以安装MikTex。

+   编辑器推荐VS Code + LaTeX WorkShop插件。
